# GermanVerbs
Learning platform for verbs in German

Online platform to study and learn verbs in German.

![Screenshot](img/screenshot.png "Screenshot")

Developed with HTML5, CSS3, Bootstrap 3.3.6 and AngularJS 1.5.0.

**Responsive design**.
